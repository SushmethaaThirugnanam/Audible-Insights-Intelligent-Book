# 📚 Audible Insights – Book Recommendation System

A Streamlit-powered interactive dashboard for exploring books, analyzing genres, and receiving personalized recommendations.  
Built with Python, pandas, scikit-learn, and NLP techniques.

---

## 🚀 Features
- **Dark-themed dashboard** with gold highlights
- **Metric cards**: total books, average rating, genre distribution
- **Genre browsing** with interactive filters
- **Recommendation cards** based on book title input
- **EDA visualizations**: rating distribution, price trends, top authors
- **Automated pipeline** for data preparation, TF-IDF feature extraction, and clustering

---

## 🛠️ Tech Stack
- **Frontend**: Streamlit
- **Backend**: Python (pandas, scikit-learn)
- **NLP**: TF-IDF vectorization
- **Clustering**: KMeans / Cosine similarity
- **Visualization**: matplotlib, seaborn, plotly
